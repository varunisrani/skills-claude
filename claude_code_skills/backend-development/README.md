# Backend Development Skills

This category contains skills for backend development, covering Python, Node.js, FastAPI, async patterns, testing, and performance optimization.

## Skills in This Category

### `async-python-patterns`
Master Python asyncio, concurrent programming, and async/await patterns for high-performance applications.

**Use when:**
- Building async APIs
- Creating concurrent systems
- Developing I/O-bound applications requiring non-blocking operations

**Key topics:**
- asyncio fundamentals
- async/await patterns
- Concurrent programming
- High-performance async applications

---

### `fastapi-templates`
Create production-ready FastAPI projects with async patterns, dependency injection, and comprehensive error handling.

**Use when:**
- Building new FastAPI applications
- Setting up backend API projects
- Need production-ready API templates

**Key topics:**
- FastAPI project structure
- Async patterns
- Dependency injection
- Error handling
- API best practices

---

### `nodejs-backend-patterns`
Build production-ready Node.js backend services with Express/Fastify, implementing middleware patterns, error handling, authentication, database integration, and API design.

**Use when:**
- Creating Node.js servers
- Building REST APIs or GraphQL backends
- Developing microservices architectures

**Key topics:**
- Express/Fastify frameworks
- Middleware patterns
- Error handling
- Authentication
- Database integration
- API design

---

### `python-packaging`
Create distributable Python packages with proper project structure, setup.py/pyproject.toml, and publishing to PyPI.

**Use when:**
- Packaging Python libraries
- Creating CLI tools
- Distributing Python code

**Key topics:**
- Package structure
- setup.py and pyproject.toml
- PyPI publishing
- Package distribution

---

## Common Use Cases

### Building a Python API
Combine these skills:
1. `fastapi-templates` - Project setup
2. `async-python-patterns` - Async operations
3. `python-testing-patterns` - Testing
4. `python-performance-optimization` - Optimization

### Building a Node.js Backend
Use:
1. `nodejs-backend-patterns` - Backend architecture
2. Consider also: `api-design-principles` from Architecture & Design

### Creating a Python Package
Use:
1. `python-packaging` - Package creation
2. `python-testing-patterns` - Test suite
3. `python-performance-optimization` - Performance tuning

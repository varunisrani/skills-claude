# DevOps & Infrastructure Skills

This category contains skills for CI/CD pipelines, cloud infrastructure, deployment automation, and cost optimization.

## Skills in This Category

### `cost-optimization`
Optimize cloud costs through resource rightsizing, tagging strategies, reserved instances, and spending analysis.

**Use when:**
- Reducing cloud expenses
- Analyzing infrastructure costs
- Implementing cost governance policies

**Key topics:**
- Resource rightsizing
- Tagging strategies
- Reserved instances
- Cost analysis
- Spending optimization

---

### `deployment-pipeline-design`
Design multi-stage CI/CD pipelines with approval gates, security checks, and deployment orchestration.

**Use when:**
- Architecting deployment workflows
- Setting up continuous delivery
- Implementing GitOps practices

**Key topics:**
- Multi-stage pipelines
- Approval gates
- Security checks
- Deployment orchestration
- GitOps

---

### `github-actions-templates`
Create production-ready GitHub Actions workflows for automated testing, building, and deploying applications.

**Use when:**
- Setting up CI/CD with GitHub Actions
- Automating development workflows
- Creating reusable workflow templates

**Key topics:**
- GitHub Actions
- Workflow automation
- CI/CD pipelines
- Reusable workflows
- Testing and deployment automation

---

### `hybrid-cloud-networking`
Configure secure, high-performance connectivity between on-premises infrastructure and cloud platforms using VPN and dedicated connections.

**Use when:**
- Building hybrid cloud architectures
- Connecting data centers to cloud
- Implementing secure cross-premises networking

**Key topics:**
- Hybrid cloud connectivity
- VPN configuration
- Dedicated connections
- Network security
- Cross-premises networking

---

### `secrets-management`
Implement secure secrets management for CI/CD pipelines using Vault, AWS Secrets Manager, or native platform solutions.

**Use when:**
- Handling sensitive credentials
- Rotating secrets
- Securing CI/CD environments

**Key topics:**
- Vault
- AWS Secrets Manager
- Secret rotation
- CI/CD security
- Credential management

---

## Common Use Cases

### Setting Up CI/CD Pipeline
Combine these skills:
1. `github-actions-templates` or `deployment-pipeline-design` - Pipeline setup
2. `secrets-management` - Secure credentials
3. Consider also: Testing skills from Code Quality & Testing

### Cloud Infrastructure Setup
Use:
1. `hybrid-cloud-networking` - Network configuration
2. `cost-optimization` - Cost management
3. `secrets-management` - Security
4. Consider also: `multi-cloud-architecture` from Architecture & Design

### Optimizing Cloud Costs
Use:
1. `cost-optimization` - Cost reduction strategies
2. Consider also: Performance optimization skills

# Code Quality & Testing Skills

This category contains skills for code review practices, debugging strategies, and error handling patterns.

## Skills in This Category

### `code-review-excellence`
Master effective code review practices to provide constructive feedback, catch bugs early, and foster knowledge sharing while maintaining team morale.

**Use when:**
- Reviewing pull requests
- Establishing review standards
- Mentoring developers
- Improving code quality

**Key topics:**
- Code review best practices
- Constructive feedback
- Bug detection
- Knowledge sharing
- Team collaboration

---

### `debugging-strategies`
Master systematic debugging techniques, profiling tools, and root cause analysis to efficiently track down bugs across any codebase or technology stack.

**Use when:**
- Investigating bugs
- Analyzing performance issues
- Troubleshooting unexpected behavior
- Root cause analysis

**Key topics:**
- Systematic debugging
- Profiling tools
- Root cause analysis
- Bug investigation
- Troubleshooting techniques

---

### `error-handling-patterns`
Master error handling patterns across languages including exceptions, Result types, error propagation, and graceful degradation to build resilient applications.

**Use when:**
- Implementing error handling
- Designing APIs
- Improving application reliability
- Building resilient systems

**Key topics:**
- Exception handling
- Result types
- Error propagation
- Graceful degradation
- Resilience patterns

---

## Common Use Cases

### Improving Code Quality
Combine these skills:
1. `code-review-excellence` - Review process
2. `error-handling-patterns` - Error handling
3. Consider also: Testing skills from Backend/Frontend categories

### Debugging an Issue
Use:
1. `debugging-strategies` - Investigation techniques
2. `error-handling-patterns` - Understanding error flow
3. Consider also: Performance optimization skills

### Building Resilient Applications
Combine:
1. `error-handling-patterns` - Error handling
2. `debugging-strategies` - Issue identification
3. Consider also: Testing skills for validation

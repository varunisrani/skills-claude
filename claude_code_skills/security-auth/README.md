# Security & Authentication Skills

This category contains skills for authentication, authorization, and security compliance.

## Skills in This Category

### `auth-implementation-patterns`
Master authentication and authorization patterns including JWT, OAuth2, session management, and RBAC to build secure, scalable access control systems.

**Use when:**
- Implementing auth systems
- Securing APIs
- Debugging security issues
- Building access control systems

**Key topics:**
- JWT (JSON Web Tokens)
- OAuth2
- Session management
- RBAC (Role-Based Access Control)
- Authentication best practices
- Authorization patterns

---

### `pci-compliance`
Implement PCI DSS compliance requirements for secure handling of payment card data and payment systems.

**Use when:**
- Securing payment processing
- Achieving PCI compliance
- Implementing payment card security measures
- Building payment systems

**Key topics:**
- PCI DSS requirements
- Payment card data security
- Compliance implementation
- Security measures
- Payment system security

---

## Common Use Cases

### Building Authentication System

### API Security

# Frontend Development Skills

This category contains skills for modern frontend development with React, TypeScript, JavaScript, and testing frameworks.

## Skills in This Category

### `frontend-dev-guidelines`
Frontend development guidelines for React/TypeScript applications with modern patterns including Suspense, lazy loading, useSuspenseQuery, file organization, MUI v7 styling, TanStack Router, and performance optimization.

**Use when:**
- Creating React components, pages, or features
- Fetching data in frontend applications
- Styling with MUI v7
- Setting up routing with TanStack Router
- Working with frontend code

**Key topics:**
- React best practices
- TypeScript patterns
- Suspense and lazy loading
- File organization
- MUI v7 styling
- TanStack Router
- Performance optimization

---

### `javascript-testing-patterns`
Implement comprehensive testing strategies using Jest, Vitest, and Testing Library for unit tests, integration tests, and end-to-end testing with mocking, fixtures, and test-driven development.

**Use when:**
- Writing JavaScript/TypeScript tests
- Setting up test infrastructure
- Implementing TDD/BDD workflows
- Testing React components

**Key topics:**
- Jest and Vitest
- Testing Library
- Unit and integration testing
- Mocking and fixtures
- Test-driven development

---

### `modern-javascript-patterns`
Master ES6+ features including async/await, destructuring, spread operators, arrow functions, promises, modules, iterators, generators, and functional programming patterns.

**Use when:**
- Refactoring legacy code
- Implementing modern patterns
- Optimizing JavaScript applications
- Learning ES6+ features

**Key topics:**
- ES6+ syntax
- Async/await
- Destructuring and spread operators
- Arrow functions
- Promises and modules
- Functional programming

---

### `typescript-advanced-types`
Advanced TypeScript type system features for building type-safe applications.

**Use when:**
- Building type-safe applications
- Working with complex TypeScript types
- Leveraging advanced type features

**Key topics:**
- Advanced type system
- Type safety
- TypeScript best practices

---

### `webapp-testing`
End-to-end testing strategies for web applications.

**Use when:**
- Implementing E2E testing
- Testing full web application flows
- Setting up automated UI testing

**Key topics:**
- End-to-end testing
- Web application testing
- Automated UI testing

---

## Common Use Cases

### Building a React Application
Combine these skills:
1. `frontend-dev-guidelines` - Component and app structure
2. `modern-javascript-patterns` - Modern JavaScript features
3. `javascript-testing-patterns` - Testing strategy
4. `webapp-testing` - E2E testing

### TypeScript Project Setup
Use:
1. `frontend-dev-guidelines` - React/TypeScript patterns
2. `typescript-advanced-types` - Advanced types
3. `javascript-testing-patterns` - Testing with TypeScript

### Testing Strategy
Combine:
1. `javascript-testing-patterns` - Unit and integration tests
2. `webapp-testing` - E2E testing

# Tools & Utilities Skills

This category contains skills for development tools, package managers, and utility skills.

## Skills in This Category

### `Skill Builder`
Create new Claude Code Skills with proper YAML frontmatter, progressive disclosure structure, and complete directory organization.

**Use when:**
- Building custom skills for specific workflows
- Generating skill templates
- Understanding the Claude Skills specification
- Creating team-specific skills

**Key topics:**
- Claude Code Skills format
- YAML frontmatter
- Progressive disclosure
- Skill structure
- Skill creation

---

### `uv-package-manager`
Modern Python package management with uv for fast, reliable dependency management.

**Use when:**
- Managing Python dependencies
- Setting up Python projects
- Need faster package management
- Working with modern Python tooling

**Key topics:**
- uv package manager
- Python dependency management
- Fast package installation
- Modern Python tooling

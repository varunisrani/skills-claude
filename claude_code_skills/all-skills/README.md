## Using Skills

### In Claude Code
Simply reference a skill by name:
```
Use the fastapi-templates skill to help me build an API
```

### Copy to Your Project
```bash
# Copy one skill
cp -r all-skills/SKILL_NAME /path/to/your-project/.claude/skills/

# Copy multiple skills
cp -r all-skills/{skill1,skill2,skill3} /path/to/your-project/.claude/skills/
```

### Browse by Category
For easier discovery, browse skills by category in the parent directory.

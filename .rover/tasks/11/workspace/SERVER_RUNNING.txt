                      TROUBLESHOOTING QUICK TIPS
================================================================================

If page doesn't load:
  1. Wait 10-15 seconds (server is initializing)

  3. Check that port 3000 is available
  4. Try opening with "localhost:3000" instead of "127.0.0.1:3000"

  1. Server is still starting
  2. Wait another 10 seconds
  3. Refresh the page

  1. Check .env file is loaded
  2. Check API credentials are set
  3. Restart the server
